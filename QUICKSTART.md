# 安卓脚本切图神器 v3.0 - 快速开始

## 启动方式

### 方式1：使用启动脚本（推荐）
```bash
# 双击运行
start.bat
```

### 方式2：使用uv
```bash
uv run python main.py
```

### 方式3：使用Python
```bash
# 先安装依赖
pip install -r requirements.txt

# 运行
python main.py
```

## 主要功能

### 1. 图片加载
- **打开文件夹**：加载整个文件夹的图片列表
- **加载本地图片**：加载单张图片
- **从剪贴板粘贴**：Ctrl+V 粘贴截图

### 2. 截图功能
- **雷电模拟器截图**：点击按钮自动截取模拟器画面
  - 支持多开（自动检测emulator-5555, 5557等端口）
  - 自动检测ADB设备
- **桌面全屏截图**：截取整个PC屏幕

### 3. ROI框选
- **框选模式**：拖拽鼠标创建选区
- **选择模式**：点击选中、拖拽移动、8方向调整大小
- **快捷键**：
  - `Delete` - 删除选中ROI
  - `Ctrl+C` - 复制ROI
  - `方向键` - 微调位置(1px)
  - `Shift+方向键` - 大幅微调(10px)

### 4. 导出功能
- **自动切图**：框选完成后自动生成切图
- **代码生成**：自动生成Auto.js和Python代码
- **批量导出**：导出JSON/XML/脚本文件

## 雷电模拟器连接

1. 启动雷电模拟器
2. 开启ADB调试（设置-关于-连续点击版本号开启开发者模式）
3. 点击"刷新"按钮检测设备
4. 点击"雷电模拟器截图"获取画面

## 输出目录

所有生成的文件保存在 `./res_output/` 目录：
- 切图文件：`{前缀}{ROI名}_{时间戳}.png`
- ROI数据：`roi_data_{时间戳}.json`
- Auto.js脚本：`auto_script_{时间戳}.js`
- Python脚本：`auto_script_{时间戳}.py`
